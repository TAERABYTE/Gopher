# 🧰 pkg/ (กล่องเครื่องมือสาธารณะ)

## 📌 หน้าที่
โฟลเดอร์นี้คือ "แพ็กเกจจิปาถะ" เป็นที่เก็บกล่องเครื่องมือ (Utility functions) สคริปต์ ตัวช่วย ที่สามารถนำไปใช้กับแพ็กเกจ/โฟลเดอร์อื่นๆ ทั่วโปรเจกต์ได้อย่างอิสระ 

ความพิเศษคือ โค้ดในนี้ **ต้องสามารถก๊อปปี้ไปใช้โปรเจกต์ Go อันอื่นๆ ในอนาคตของคุณได้ทันที** โดยที่ไม่ต้องแก้โค้ด!! 

## ❌ ข้อห้าม
- **ห้ามเรียกใช้ `internal/` หรือ `domain/`** : แพ็กเกจในนี้ต้องทำงานได้ด้วยตัวมันเอง ห้ามทำการ Import โมเดลหรือฟีเจอร์ใดๆ ของแอปพลิเคชัน (มันห้ามรู้ว่าแอปนี้พัฒนาให้บริษัทอะไร ทำแอป Notes หรือระบบอะไร)

## 💡 ตัวอย่าง
- **`pkg/response`** : ท่ามาตรฐานสำหรับการเขียน JSON Format กลับส่ง User เราไม่ต้องเขียน `json.NewEncoder()` ซ้ำไปซ้ำมา
- **`pkg/jwt`** : ระบบ Generate และ ถอดรหัส JWT Token... เราส่งตัวเลขรับตัวอักษร วันหน้าที่เราไปทำระบบแชท เราก็โยนโฟลเดอร์นี้ไปใช้ต่อได้ทันที
