# 🏁 cmd/ (Command)

## 📌 หน้าที่
โฟลเดอร์นี้คือ **จุดเริ่มต้น (Entry Point)** ของตัวแอปพลิเคชัน 

หน้าที่หลักของมันคือ **การประกอบร่าง (Wiring/Dependency Injection)** เท่านั้น 
ไฟล์ `main.go` ในนี้จะทำหน้าที่โหลด Config, ต่อ Database, สร้าง Repository, Usecase, และ Handler จากนั้นจับคู่ทุกอย่างเข้าด้วยกันแล้วสั่งรัน HTTP Server

## ❌ ข้อห้าม
- **ห้าม** เขียน Business Logic (การประมวลผลข้อมูล) ไว้ในนี้เด็ดขาด
- **ห้าม** เขียน HTTP Route ไว้ในนี้แบบตรงๆ (ให้ไปเขียนใน `delivery`)

## 💡 ตัวอย่างการทำงาน
เหมือน **ผู้จัดการร้าน (main.go)** ที่มีหน้าที่จ้างเด็กเสิร์ฟ (Handler), จ้างพ่อครัว (Usecase), จ้างคนซื้อของ (Repo), เช่าครัว (DB) แล้วจัดให้ทุกคนมาทำงานร่วมกัน จากนั้นก็สั่ง "เปิดร้าน!" (`server.ListenAndServe()`)
