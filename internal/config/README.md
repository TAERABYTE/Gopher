# ⚙️ config/

## 📌 หน้าที่
รับผิดชอบในการดึงค่าการตั้งค่าสภาพแวดล้อมต่างๆ เช่นค่าจากไฟล์ `.env` หรือ OS Environment Variables นำมาจับใส่ใน `Struct` เพื่อให้ส่วนอื่นของแอปพลิเคชันเอาไปใช้งานต่อได้ง่ายและปลอดภัย โดยไม่ต้องมารันคำสั่ง `os.Getenv()` กระจัดกระจายทั่วทั้งโปรเจกต์

## 💡 ตัวอย่างการทำงาน
แทนที่เราจะจำว่ารหัสผ่าน Database ต้องโหลดจากกุญแจชื่อ `DB_DSN` เราก็ให้โฟลเดอร์นี้ทำการอ่านมาให้รอบเดียวตอนเปิดโปรแกรม แล้วเก็บไว้ในตัวแปร `Config` เวลานำไปใช้ก็เรียก `cfg.DB_DSN` ได้เลย
