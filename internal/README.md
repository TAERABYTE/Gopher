# 🏢 internal/

## 📌 หน้าที่
โฟลเดอร์นี้เป็นฟีเจอร์เฉพาะของภาษา Go 
โค้ดโฟลเดอร์อื่นๆ ที่ไม่ได้อยู่ภายใต้โปรเจกต์นี้ **จะไม่สามารถ Import โค้ดในโฟลเดอร์นี้ไปใช้ได้** 

มันคือ **ซอร์สโค้ดเฉพาะทางของโปรเจกต์เรา** (Private Code) เพื่อป้องกันไม่ให้โปรเจกต์อื่นมาดึงโค้ดแกนหลักของเราไปใช้แบบมั่วซั่ว โค้ดทั้งหมดที่ทำมาหากินของ Backend เราจะถูกซ่อนอยู่ในนี้

ภายในจะแบ่งโครงสร้างย่อยแบบ **Clean Architecture**: delivery -> usecase -> repository -> domain
- `config/` - ตัวจัดการการตั้งค่า (.env)
- `delivery/` - ประตูรับคนไข้ (HTTP)
- `domain/` - โครงสร้างข้อมูลหลัก (พระเอกของงาน)
- `repository/` - ห้องเก็บประวัติ (Database)
- `usecase/` - หมอวินิจฉัยโรค (Business Logic)
