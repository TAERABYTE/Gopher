# 🌍 delivery/ (ประตูก้าวสู่โลกกว้าง)

## 📌 หน้าที่
รับผิดชอบติดต่อสื่อสารกับ "โลกภายนอก" 
ในโปรเจกต์นี้คือส่วนที่รับ Request แบบ HTTP และจัดการให้ได้ Response ออกมา 

โฟลเดอร์นี้รับหน้าที่:
1. **Routing (`router.go`)** : จับคู่เส้นทาง (Paths) เช่น `/api/notes` ไปยังฟังกชันที่ต้องทำ
2. **Handler/Controller (`handler/`)** : เป็นคนรับ HTTP Request (`*http.Request`) ดึงเอา JSON ที่ส่งมา -> แกะ JSON เปลี่ยนเป็น Struct ของภาษา Go -> ตรวจสอบว่าใส่ข้อมูลครบไหม -> ส่งต่อไปให้ `Usecase` เป็นคนตัดสินใจ
3. **Middleware (`middleware/`)** : ด่านตรวจคนเข้าเมือง เช่นตัวตรวจจับและถอดรหัส `JWT Token` ว่าก่อนเข้า Route ควบคุมจะต้องมีพาสปอร์ตจริงเท่านั้น

## ❌ ข้อห้าม
- **ห้ามใส่ Business Logic** : (เช่น เอา Username มาพึ่งคำนวณ Hash หรือแปลงค่า Password ที่นี่) ให้โยนทั้งหมดไปให้ `Usecase` สั่งการแทน
- **ห้ามคุยกับ Repository โดยตรง** : ต้องคุยผ่าน `Usecase` เท่านั้น (Handler -> Usecase -> Repository)

## 💡 ตัวอย่าง
ไฟล์ `note_handler.go` พอมียิง `/api/notes` เข้ามาด้วยท่า `POST`:
1. มันจะรับ Request เข้ามา
2. จับคู่แปลงกล่อง JSON `{ "title": "hello" }` ออกมา
3. โยนไปให้ `NoteUsecase` บอกว่า "ช่วยไปเซฟ Note อันนี้ให้สิ"
4. Usecase ส่งคำตอบกลับมาว่าได้หรือไม่ได้ ถ้าได้ Handler จะเปลี่ยนมันกลับเป็น JSON ตอบกลับให้ Client
